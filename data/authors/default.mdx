안녕하세요! 프론트엔드 개발자 김석진입니다.

현재 **골프존**에서 내부 백오피스와 모바일 앱 웹뷰 서비스를 개발하고 있습니다. 캠페인 발송 도구, 통계 시스템(GSTAT) 리팩토링, 플러스샵·혜택존 등 골프존 앱의 핵심 기능을 Vue 3 + Vite 기반 모던 스택으로 개선하며, 레거시 청산과 번들 성능 최적화에 집중하고 있습니다.

퍼블리싱으로 시작해 점차 프론트엔드 개발 영역으로 확장해 왔습니다. 민팃·GS 자이 모바일 웹에서 반응형 마크업과 키오스크 UI를 다루고, 배움컴퍼니 소개페이지에서는 GSAP 기반 인터랙티브 모션을 구현했습니다. 이후 **Papaya LMS**, **키움히어로즈 어드민** 등 Vue 2 + TypeScript 백오피스를 설계하며 대규모 권한 제어·비즈니스 로직 모듈화 경험을 쌓았고, **Java-metacommerce**에서는 글로벌 e-commerce 솔루션의 SSR/CSR 하이브리드 아키텍처, 다국어(i18n), 해외 결제 연동을 주도했습니다.

최근에는 **Next.js 15 · React 19** 스택으로 AI 기반 교육 플랫폼(Graphit)과 팀 매칭 솔루션(OPSM)을 구축하며, Vercel CI/CD와 Husky 기반 코드 품질 자동화 파이프라인도 함께 설계했습니다.

주로 다루는 영역은 다음과 같습니다.

- **백오피스·어드민**: 권한 제어, 복잡한 폼·그리드, 대시보드
- **모바일 웹뷰**: 하이브리드 앱 내 인터랙션·모션 최적화
- **e-commerce**: 주문·셀러 관리, 결제·다국어 대응
- **레거시 현대화**: Vue 2 → Vue 3, Webpack → Vite 마이그레이션

기술을 채우는 것만큼이나, 비우고 새로 배우는 과정(Unlearn)을 중요하게 생각합니다. 새로운 스택을 도입할 때도 기존 코드와 팀의 맥락을 존중하면서, 더 나은 개발 경험과 사용자 경험을 함께 만들어 가고 있습니다.
