# Golfzon - Benefit Zone (골프존 혜택존 리뉴얼)

골프존 모바일 앱 사용자의 리텐션을 제고하고, 다양한 혜택과 즐길 거리를 한눈에 직관적으로 제공하도록 혜택존의 구조와 시각 모션을 리뉴얼한 웹뷰 기반 프로젝트입니다.

## 🛠️ 기술 스택 (Spec)

- **Node.js**: `22.14.0`
- **Framework**: Vue.js `3.2.13`
- **State Management**: Pinia
- **Platform**: 모바일 하이브리드 앱 웹뷰 환경

## 🚀 주요 역할 및 수행 내용

### 1. 혜택존 전면 개편

- 기존의 나열식 텍스트/리스트 위주의 투박한 구조에서 탈피하여 각 브랜드 및 미션 이벤트 성격에 적합한 섹션 카드 중심 레이아웃으로 UI를 리디렉션하여 가독성과 사용자 참여율을 극대화했습니다.

### 2. 스윙만보기 연동 스크롤 트랜지션

- 만보기 소개 페이지에서 스크롤 흐름에 따라 그래픽과 텍스트 카드가 리얼타임으로 교차 페이드하는 인터랙티브 가이드를 설계했습니다.
- **Intersection Observer API**와 Vue의 기본 `<transition>`을 결합하여, 모바일 단말기 자원을 최소화하면서도 정밀한 스크롤 연동 페이드 애니메이션을 구축했습니다.

### 3. 마일리지 미션 팝업 및 오늘 하루 보지 않기 구현

- **오늘 하루 보지 않기**: 로컬 스토리지 데이터 관리를 바탕으로 매일 하루 단위 팝업 기한을 계산하는 유틸 기능을 모듈화하여, 사용자의 앱 이용 피로도를 낮추었습니다.
- **마일리지 팝업 애니메이션**: 미션을 클리어한 사용자가 골프존 마일리지를 획득할 때, 생동감 넘치는 팝업의 모션 연출(Vue transition)을 지원하여 성취감(Gamification)을 자극했습니다.
