# Golfzon - GSTAT (골프존 통계 시스템 리팩토링)

사용자 중심의 직관적인 데이터 분석을 위해 기존의 노후화된 통계 조회 백오피스 시스템을 최신 모던 프론트엔드 아키텍처로 완전히 이관 및 재구축한 프로젝트입니다.

## 🛠️ 기술 스택 (Spec)

- **Framework**: Vue.js `3.4.34` (Composition API)
- **State Management**: Pinia
- **Build Tool**: Vite `5.3.1`
- **UI Library**: PrimeVue (Sakai-vue 테마 기반)
- **Styling**: Tailwind CSS
- **Language**: JavaScript

## 🚀 주요 역할 및 수행 내용

### 1. 전면적인 레거시 마이그레이션

- `vue2` (Option API) + `vue-cli` + `vuex` + `admin-lte` 기반으로 운영되던 무겁고 복잡한 기존 프로젝트를 **Vue 3 (Composition API) + Vite + Pinia + PrimeVue** 조합으로 완벽하게 전환하였습니다.
- 번들 사이즈 및 런타임 오버헤드를 줄여 데이터가 많은 통계 화면의 초기 렌더링 성능과 화면 전환 속도를 획기적으로 상승시켰습니다.

### 2. 고도의 사용자 상호작용 그리드 구현

- **컬럼 드래그 앤 드롭 조회**: `vue-dndrop` 라이브러리를 도입하여, 통계 어드민 사용자가 분석할 데이터 목록의 컬럼 순서를 실시간 마우스 드래그로 변경하고 조회할 수 있도록 개선하였습니다.
- **스프레드시트 방식 대용량 데이터 입력**: `vue3-excel-editor`를 도입 및 커스터마이징하여, 엑셀과 같은 행/열 단위 입력 인터페이스를 제공함으로써 수기로 입력하는 다량의 데이터를 관리자가 빠르고 오차 없이 저장할 수 있도록 지원했습니다.

### 3. AI 기반 개발 파이프라인 연계

- Cursor IDE의 AI 개발 에이전트를 적극 사용하여 프로젝트의 아키텍처 규범(`.mdc` ruleset)을 강제 적용해 코드 가독성과 재사용성을 큰 폭으로 보장했습니다.
