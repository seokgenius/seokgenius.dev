# Golfzon - Plusshop (골프존 모바일 플러스샵 유지보수)

골프존 공식 모바일 앱 내에서 모바일 상품권, 스크린 아이템 등을 구매하고 선물할 수 있는 쇼핑 서비스인 '플러스샵'의 비즈니스 요건 대응 및 신규 인터페이스 고도화를 수행했습니다.

## 🛠️ 기술 스택 (Spec)

- **Node.js**: `16.20.0`
- **Framework**: Vue.js `3.0.0` (Option API)
- **State Management**: Vuex `4.0.0`
- **Platform**: 모바일 하이브리드 앱 웹뷰 환경

## 🚀 주요 역할 및 수행 내용

### 1. 하이브리드 웹뷰 인터랙션 성능 안정화

- 웹뷰 특유의 터치 반응 지연 현상을 억제하고 모바일에 가장 부드러운 스크롤 인터랙션을 위해 마운트 주기 단계에서 불필요한 재렌더링을 억제하였습니다.

### 2. 홀인원볼 자동결제 UI 고도화

- 사용자가 홀인원볼 상품의 자동결제 옵션(라디오 버튼 등)을 토글하거나 수정할 때, 즉각적이고 매끄럽게 밑에서 솟아오르는 슬라이드 모달 및 버튼 상태 값의 정밀 제어를 수행했습니다.
- 입력 선택 변화에 상응하여 버튼의 문구, 활성화 상태 변경 및 슬라이드 UI 간의 전환 트랜지션을 모바일 네이티브 앱 수준으로 매끄럽게 연동되도록 Vue reactivity를 설계하였습니다.
