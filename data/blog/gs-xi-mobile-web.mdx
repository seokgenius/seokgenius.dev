# GS Xi mobile web (GS 자이 모바일 통합시스템 퍼블리싱)

대한민국 대표 프리미엄 아파트 브랜드인 GS Xi의 입주민 전용 모바일 웹 인터페이스를 정교하게 구현한 웹 마크업 및 인터랙션 개발 프로젝트입니다.

## 🛠️ 기술 스택 (Spec)

- **Markup & Styling**: HTML5, SCSS (시맨틱 마크업 가이드 준수)
- **Scripting & Interaction**: JavaScript, jQuery
- **Platform**: 다양한 모바일 기기 브라우저 대응

## 🚀 주요 역할 및 수행 내용

### 1. 고밀도 시맨틱 마크업 및 웹 접근성 표준 설계

- 입주 정보, 하자 신청 등 텍스트 가독성이 대단히 중요한 구조에 알맞은 HTML5 시맨틱 태그 구조를 직접 설계하고, 기기 해상도에 유기적으로 가변 반응하는 SCSS 레이아웃을 작성했습니다.
- 스크린 리더와 모바일 포커싱 흐름을 감안하여 웹 접근성 규칙을 세밀하게 준수했습니다.

### 2. jQuery 기반 고성능 모바일 UI 라이브러리 커스텀 구현

- 기성 라이브러리의 무거운 오버헤드를 막기 위해 jQuery를 이용하여 입주민용 터치 프렌들리 아코디언 메뉴, 슬라이드 토글러, 스크롤 락 기능이 탑재된 입체적 모달 팝업 등을 맞춤 설계하여 구현하였습니다.
- 모바일 웹뷰에서 터치 반응 속도를 끌어올리기 위해 스타일 및 모션 하드웨어 가속(GPU) CSS 처리를 병행했습니다.
