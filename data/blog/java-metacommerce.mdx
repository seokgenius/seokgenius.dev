# Java-metacommerce (E-commerce 솔루션 구축)

메타커머스(Meta-commerce)의 신규 글로벌 e-commerce 솔루션 구축 프로젝트입니다. 백오피스(어드민) 관리 시스템부터 다국어 대응, SNS 로그인, 가격 비교(VS pick) 및 해외 결제 모듈 연동까지 아우르는 풀스택 프론트엔드/미들웨어 시스템을 구축했습니다. SSR(서버 사이드 렌더링)과 CSR(클라이언트 사이드 렌더링)을 결합하여 웹 초기 로딩 속도와 SEO 최적화를 동시에 달성한 Vue 3 기반 모던 웹 어플리케이션입니다.

배포 데모 사이트: [demo-j.meta-commerce.co.kr](https://demo-j.meta-commerce.co.kr/) (글로벌: `demo-jg.meta-commerce.co.kr`)

---

## 🛠️ 기술 스택 (Spec)

- **공통 Spec**: Node.js `18.16.0`, TypeScript `4.9.5`
- **Client Framework**: Vue 3 `3.4.21` (Vite `5.1.4` SSR + CSR 하이브리드 구조)
- **State Management**: Pinia `2.1.7` (기존 Vuex `4.1.0`에서 전면 마이그레이션)
- **Routing & Validation**: Vue Router `4.3.0`, Vee-Validate `4.12.5`
- **HTTP Client & Multi-language**: Axios `1.6.7`, Vue-i18n `9.9.1`
- **Server Middleware**: Express `4.17.21` (PM2 `5.3.1` 프로세스 관리, Winston `3.12.0` 로깅)
- **Code Quality**: ESLint `8.57.0`, Prettier `2.8.8`
- **Third Party Libraries**: Chart.js (가격 비교 그래프 시각화), TinyMCE `6.8.3` (Rich Text Editor), Eximbay (해외 결제 API 연동)

## 🚀 주요 역할 및 수행 내용

### 1. 백오피스(어드민) 관리 시스템 설계 및 구축

메타커머스 솔루션의 핵심인 어드민 시스템의 다차원 비즈니스 로직을 구축했습니다.

- **셀러(업체) 권한 및 기본설정 관리**: 개별 입점 브랜드 및 셀러들이 상품, 결제 조건, 정산 방식 등을 직접 통제할 수 있도록 격리된 권한 체계와 파트너 전용 백오피스 UI를 개발했습니다.
- **주문 생애 주기 관리 파이프라인**: 입금 확인부터 시작해 배송 준비, 배송 중, 배송 완료 단계는 물론 교환/반품/취소와 같은 클레임 관리 프로세스 전체를 아우르는 정밀한 주문 처리 워크플로우 화면을 설계 및 탑재했습니다.
- **Rich Text Editor 통합**: 공지사항 및 상세 페이지 편집을 위해 TinyMCE 에디터를 이식하고, 다중 이미지 비동기 업로드 모듈과 vee-validate 기반 폼 입력 검증 규칙을 안전하게 연동했습니다.

### 2. 다국어 글로벌 대응 및 해외 결제(Eximbay) 모듈 연동

글로벌 비즈니스로의 확장을 위해 다국어 국제화(I18n) 규격과 결제 보안 모듈을 구축했습니다.

- **국가별 다국어 및 반응형 레이아웃 처리**: `vue-i18n`을 적용하여 회원가입, 로그인, 마이페이지(회원 정보 및 주문 관리) 영역의 다국어 번역을 연동했습니다. 특히 영어, 중국어, 일어 등 언어별 텍스트 길이에 대응하도록 UI 레이아웃의 여백과 글자 줄바꿈 규격을 고도화했습니다.
- **Eximbay 해외 결제 API 연동**: 글로벌 결제 처리를 위해 Eximbay 결제 모듈의 암호화 규격(Hash Key 검증 등)에 부합하도록 프론트엔드와 Express 미들웨어 간의 시큐어 통신을 구현하고, 통신 실패나 한도 초과 등 결제 시나리오별 예외 피드백을 강화했습니다.

### 3. 클라이언트 기능 고도화 (SNS 연동 및 Chart.js 가격 비교)

소비자의 편의성과 풍부한 인터랙션을 제공하는 클라이언트 페이지를 개발했습니다.

- **VS pick 가격 추이 시각화**: 비교검색 페이지에서 사용자가 관심 있는 여러 상품의 날짜별 가격 변동 흐름을 직관적으로 비교할 수 있도록 Chart.js를 연동하고, 모바일 및 데스크톱 뷰포트에 적응하는 반응형 차트 레이아웃을 완성했습니다.
- **OAuth 간편 로그인**: 카카오, 네이버, 구글 등 주요 소셜 로그인 흐름을 마이페이지 통합 인증 로직과 결합하여 회원 가입 및 결제 전환율을 개선했습니다.

### 4. Vite SSR/CSR 하이브리드 아키텍처 수립 및 상태 관리 현대화

- **SSR/CSR 하이브리드 구축**: 초기 로딩 속도와 웹 최적화 성능을 끌어올리기 위해 Vite 환경에서 SSR(서버 사이드 렌더링)을 결합하여 SEO 친화적인 static html을 먼저 렌더링하고, 이후 hydration 과정을 통해 매끄러운 CSR 싱글 페이지 동작을 보장하는 파이프라인을 구축했습니다.
- **Pinia 전역 스토어 마이그레이션**: 기존에 사용하던 Vuex 4 구조를 100% TypeScript 타입 추론과 모듈화가 용이한 Pinia 스토어로 마이그레이션하여 런타임 번들 크기를 줄이고 코드 관리 생산성을 대폭 높였습니다.
- **Express & PM2 인프라 레이어**: 프론트엔드와 백엔드 API 간 미들웨어 및 서버 사이드 렌더링을 호스팅하는 Express 서버를 구축하고, PM2 클러스터링 모드를 통해 서버 장애 시 즉각 복구되는 무중단 배포 가이드라인을 이식했습니다.
