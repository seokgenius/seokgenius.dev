# Kiwoom heroes Admin (키움히어로즈 관리자 시스템)

프로 야구단 키움히어로즈의 공식 구단 모바일 애플리케이션에 적재되는 공지사항, 경기 기록, 티켓 발송 및 선수 데이터 정보 등을 안정적으로 관리하고 편집하는 백오피스 개발 프로젝트입니다.

## 🛠️ 기술 스택 (Spec)

- **Framework**: Vue.js `2.x` (TypeScript 구성)
- **State Management**: Vuex (`vuex-module-decorators` 및 `vuex-class` 활용)
- **Styling**: SCSS (직접 마크업 퍼블리싱 담당)
- **Language**: TypeScript

## 🚀 주요 역할 및 수행 내용

### 1. 동적 레이아웃 템플릿 아키텍처 설계

- 다양한 종류의 메뉴와 데이터 폼 레이아웃을 효율적으로 렌더링하기 위해, Vue의 동적 컴포넌트(`<component :is="...">`) 스펙을 응용하여 유연한 어드민 레이아웃 구조를 완성하였습니다.
- 새로운 양식 메뉴가 추가될 때 공통 스펙만 상속받아 쉽게 뷰를 전개하도록 코드 상속 모델을 정립했습니다.

### 2. 라우터 네비게이션 가드 기반의 페이지 권한 제어

- 다수의 부서 및 파트너사가 공동 사용하는 어드민 특성에 맞춰, **Route Meta** 값에 허용 권한 리스트를 내장하고, 라우터 네비게이션 가드(`beforeEach`)를 통해 부적절한 사용자 접근을 원천 차단하는 보안 환경을 구축했습니다.

### 3. 실시간 다차원 검색 필터링 및 모션 트랜지션

- 수만 건의 구단 기록을 가볍게 선별할 수 있는 다중 입력 필터 모듈을 설계하고, UI 조작 단계에서 복잡함을 덜기 위해 Vue의 내장 Transition 모션 효과를 얹어 부드러운 토글 및 모달 레이아웃을 직접 퍼블리싱했습니다.
