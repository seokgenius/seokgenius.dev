# Papaya Web (파파야 웹 개발)

TypeScript 기반의 강력한 타입 정의와 안정적인 폼 검증 모듈을 갖춰 정밀한 웹 관리를 보조하는 프론트엔드 프로젝트입니다.

## 🛠️ 기술 스택 (Spec)

- **Framework**: Vue.js `2.6.11` (Vue Router `3.2.0`, Vue Class Component `7.2.3`, Vue Property Decorator `8.4.2`)
- **State Management**: Vuex `3.4.0` (vuex-class `0.3.2`, vuex-module-decorators `1.0.1`)
- **UI Library**: Vuetify `2.4.9`
- **Form Validation**: Vee-Validate `3.4.5`
- **Language**: TypeScript `3.9.3`
- **Styling**: SCSS

## 🚀 주요 역할 및 수행 내용

### 1. 객체 지향형 Vue 클래스 컴포넌트 모델 수립

- `vue-class-component` 및 `vue-property-decorator` 데코레이터를 적극 수용하여, 가독성이 높고 상속이 용이한 객체지향 형태의 컴포넌트 코드베이스를 수립했습니다.
- TypeScript의 엄격한 모드를 유지하며 상태 및 메서드에 명확한 인터페이스를 부여하여 잠재적 버그를 사전 배제했습니다.

### 2. Vuetify 및 SCSS 스타일 연동 고도화

- Vuetify의 기본 프레임에 맞춰 SCSS 변수 값을 유기적으로 재매핑하고, 복잡한 비즈니스 폼과 어드민 그리드를 신속하게 렌더링할 수 있는 UI 템플릿들을 다수 설계했습니다.

### 3. Vee-Validate 기반 유효성 피드백

- 여러 단계로 나뉘는 사용자 입력 폼에 대해 `Vee-Validate`의 사전 유효성 검증 시스템을 탑재하여 에러 피드백을 실시간 제공하고, 올바른 데이터 송신만을 허용하는 정밀 흐름 제어를 지원했습니다.
