# SLW - OPSM

강력한 프론트엔드 인프라와 상태 제어 흐름을 가진 Next.js 기반 웹 플랫폼 솔루션 프로젝트입니다.

## 🛠️ 기술 스택 (Spec)

- **Node.js**: `22.16.0`
- **Framework**: Next.js `15` + React `19`
- **State Management**: Zustand
- **Styling**: Tailwind CSS + Shadcn/ui + SCSS
- **Deployment**: Vercel (자동 배포 연계)
- **Linter/Formatter**: ESLint (Airbnb) + Prettier + Husky (Git GitHooks)
- **배포 웹사이트**: [slw-opsm.vercel.app](https://slw-opsm.vercel.app/)

## 🚀 주요 역할 및 수행 내용

### 1. 차세대 React 19 및 Next.js 15 기능 튜닝

- 비동기 상태 핸들링과 UI 성능 향상을 위해 React 19의 모던 컨셉과 Next.js 15 서버 최적화를 탑재하였습니다.
- CSS 모듈 구조와 글로벌 Tailwind 규칙을 병합하여 비정상 디자인 깨짐 및 빌드 타임 스타일 번들 낭비를 적극 방지했습니다.

### 2. Zustand 기반의 명확한 상태 파이프라인

- 다중 뷰포트 상태와 실시간 변경되는 대규모 양식(Form) 데이터를 효율적으로 상태 저장하고 관리하기 위해 Zustand를 도입하였습니다.
- 상태 공유 아키텍처를 단방향으로 수립하여 불필요한 UI 디바운싱을 없애고 즉각적인 응답성을 획득했습니다.

### 3. Vercel 배포 파이프라인 통합

- 소스 코드 체크인 시 즉시 패키징 오류를 잡아내는 린팅 및 Vercel 배포 파이프라인을 온보딩하여 애자일한 스프린트 업데이트를 실현했습니다.
