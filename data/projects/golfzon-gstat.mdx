# Golfzon - GSTAT

사용자 중심의 직관적인 데이터 분석과 현대적인 관리 환경을 위해 기존의 노후화된 통계 조회 백오피스 시스템을 최신 모던 프론트엔드 아키텍처로 완전히 이관 및 재구축한 프로젝트입니다.

---

## 🛠️ 기술 스택 (Spec)

기존 레거시 스택(Vue 2, Vuex, AdminLTE)을 전면 리팩토링하여 로딩 성능과 유지보수성을 극대화하였습니다.

| 항목                      | Legacy (`gstat-new`) | Modernized (`gstat-renewal`)                  |
| :------------------------ | :------------------- | :-------------------------------------------- |
| **Framework**             | Vue 2 (Options API)  | **Vue 3 (Composition API, `<script setup>`)** |
| **State Management**      | Vuex                 | **Pinia**                                     |
| **UI Template**           | AdminLTE             | **Sakai Vue (PrimeVue 기반)**                 |
| **Styling**               | Bootstrap            | **Tailwind CSS + SCSS**                       |
| **Build Tool & Language** | Webpack, JavaScript  | **Vite, JavaScript**                          |

---

## 🚀 주요 역할 및 수행 내용

### 1. 현대적인 아키텍처로의 전환 및 리팩토링

- 레거시 코드인 `vue-cli` + `vuex` + `admin-lte` 기반의 구조를 **Vite + Vue 3 (Composition API) + Pinia + PrimeVue** 조합으로 완전히 이관하였습니다.
- 성능 병목을 유발하던 무거운 라이브러리를 제거하고 런타임 오버헤드를 줄여 대용량 통계 데이터를 다루는 본 시스템의 초기 로딩 및 화면 전환 속도를 획기적으로 개선했습니다.

### 2. 고도의 데이터 시각화 및 스마트 분석 기능 구현

- **실시간 비즈니스 지표 시각화**: 일/주/월/년 단위의 핵심 비즈니스 지표를 다차원 테이블 및 **Chart.js** 기반 차트로 시각화하여 비즈니스 현황을 한눈에 조망할 수 있는 통합 대시보드를 구축했습니다.
- **개인화 즐겨찾기**: 개별 관리자가 자주 모니터링하는 핵심 지표를 따로 모아 빠르게 확인할 수 있는 맞춤형 즐겨찾기 공간을 제공합니다.
- **다차원 피벗 테이블**: 행(Row), 열(Column), 값(Value) 필드를 관리자가 직접 정의하고 자유롭게 필터링할 수 있는 피벗 테이블 기능을 개발하여 유연한 데이터 분석 환경을 마련했습니다.
- **SQL 성능 분석 도구**: 조회 쿼리의 실시간 성능 메트릭(API 응답 시간, 파싱 속도 등)을 시각화하여 개발자 및 시스템 운영팀이 데이터 정합성 검증 및 병목 지점 최적화를 지원할 수 있도록 설계했습니다.

### 3. 데이터 추출 및 보안 관리 워크플로우 체계화

- **맞춤형 데이터 추출 프로세스**: 개인정보추출, SO(매장/가맹점)정보추출, 일반데이터추출 등의 업무적 필요에 따른 데이터 추출 신청 창구를 일원화했습니다.
- **결재 시스템 연동**: 신청된 데이터 요청에 승인 및 반려 단계를 거치는 안전하고 투명한 관리 워크플로우를 도입하여 비즈니스 데이터 유출 위험을 줄였습니다.

### 4. 스프레드시트형 앱 트래킹 및 일괄 작업 생산성 개선

- **Firebase 태그 관리 고도화**: 앱 내에서 수집되는 무수한 이벤트 태그(Page ID, Action, Label 등)를 화면에서 편리하게 관리할 수 있도록 스프레드시트 인터페이스(`vue-excel-editor` 기반)를 적용했습니다.
- **엑셀 연동 및 일괄 작업**: 행 삽입/삭제, 복사-붙여넣기 및 엑셀 업로드/다운로드 기능을 제공하여 수만 건에 달하는 앱 트래킹 코드를 수동 작업하는 비효율을 해소했습니다.

### 5. 세밀한 권한 통제 시스템 구축 (Admin Console)

- **TreeTable 기반 사용자 권한 관리**: 어드민 사용자의 접근 범위를 보다 세밀하게 설정할 수 있도록 계층형 구조의 TreeTable을 적용했습니다.
- 이를 통해 매트릭스 조회, 스마트분석, 데이터 추출 신청, 앱트래킹 등 메뉴별 접근 및 조작 권한을 직관적이면서도 철저하게 제어할 수 있도록 구현했습니다.

### 6. AI 기반 개발 파이프라인 연계

- Cursor IDE의 AI 개발 에이전트를 적극 사용하여 프로젝트의 아키텍처 규범(`.mdc` ruleset)을 강제 적용해 코드 가독성과 재사용성을 큰 폭으로 보장했습니다.
